# Ahmed Akram Portfolio - Legacy V2

This directory archives the complete source of the second portfolio design: the
premium-dark engineering drawing set with the interactive Dispatch.

It was copied before the 2026 redesign and is intentionally kept unchanged
apart from this archive note. Generated folders such as `node_modules` and
`.next` are excluded.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
